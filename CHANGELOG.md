All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-04-26

### Added
- `CLAUDE.md` — codebase system prompt template
- `AGENTS.md` — agentic workflow boundaries
- `SPEC_TEMPLATE.md` — pre-build planning prompt
- `PR_TEMPLATE.md` — post-build PR description prompt
- `WORKFLOW.md` — full pattern from idea to merged code
- `setup.sh` — interactive setup script (8 questions, generates custom CLAUDE.md)
- Example configurations for 8 project types:
  - `rest-api` — backend API
  - `nextjs-app` — full-stack Next.js application
  - `mcp-server` — MCP server for agentic workflows
  - `pseo` — programmatic SEO platform
  - `aeo` — answer engine optimization (schema engineering)
  - `data-pipeline` — ETL data pipeline
  - `conversational-ai` — multi-channel AI agent platform
  - `solo-builder` — minimal setup for individual developers
